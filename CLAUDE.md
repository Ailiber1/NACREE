# NACRÉE プロジェクト CLAUDE.md

## ハーネス構成
**大規模構成: Planner + Generator + Evaluator（3役割）/ 時間差合流**
- Planner: メインエージェント
- Generator: サブエージェント（Phase間でリセット）
- Evaluator: Generatorとは別のサブエージェント

## デプロイ先
Vercel（無料枠・個人利用）— API Routes + Server Components使用のため
デプロイURL: https://nacree.vercel.app

## セキュリティプロファイル
**選定: Tier 4 — Commerce（28項目 / 118点満点）**
判定理由: Stripe決済あり + Supabase認証 + 顧客データ管理 + 管理者権限分離

### 適用項目
S-01〜S-28 の全項目を適用

### 合格ライン
118点 × 80% = 94.4点 → **95点以上で合格**

## デザイン判断
- **Tone**: serene-luxury（静謐な高級感）— 高級ホテルのスパのような落ち着きと上品さ
- **Differentiation**: 「開いた瞬間にサロンの世界観に引き込まれ、3タップで予約完了する」
- **Typography**: Noto Serif JP（見出し）+ Inter（数字・時間）+ Noto Sans JP（本文）
- **Color**: Warm Ivory #FAF6F1 / Deep Charcoal #1A1A1A / Antique Gold #B8956A / Sage Green #8FA98F
- **Motion**: 予約確定時のみフェードイン。スクロールアニメーション禁止
- **Spatial**: セクション間80px以上、カード内padding 32px以上
- **Texture**: 高品質背景写真 + オーバーレイで統一感

### AIスロップ排除
- 紫グラデーション禁止
- rounded-2xl以上の角丸禁止（rounded-lg = 8pxまで）
- 3カラム均一カードレイアウト禁止
- 「Welcome to...」的ヒーローセクション禁止
- 意味のないアイコン羅列禁止
- グラデーションボタン禁止
- パーティクル・星空・浮遊要素禁止

## 技術スタック
| 技術 | 用途 |
|------|------|
| Next.js 15 (App Router) | フロントエンド + API |

| Stripe | 決済 |
| Cloudflare Pages | ホスティング |

| Resend | メール通知 |
| Recharts | グラフ |

```
bookings          — 予約
booking_payments  — 決済情報
reviews           — レビュー
business_settings — 営業設定
```
